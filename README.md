# Проект: Фильмопоиск
------
Интерактивная страница, куда можно добавлять фильмы и удалять их.
------
В работе применены:
* передовые свойства "Флексбокс"
* прогрессиная сетка типа "Грид"
* запрограмированно на специальной библиотеке "Реакт" для языка "ДжаваСкрипт"
